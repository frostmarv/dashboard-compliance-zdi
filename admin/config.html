Cara Mengubah Status Form
  1. Klik tombol "Buka" atau "Tutup" di bawah
  2. Salin kode JSON yang muncul
  3. Buka file admin/config.json di GitHub
  4. Paste kode JSON tersebut dan commit
  5. Form akan terupdate otomatis dalam 1-2 menit
